# kaliMantan
**kaliMantan** is a math game for multiplication adopts from the *ladder snake game* or *ular tangga*. This game tells about crossing **Kalimantan Island** with mathematics symbols (Ã—) times. To move the player based on the number of questions / Math Quest (Multiplication) that can be solved on Turn Time that set. How the first player that can go to finish is a Winner Player.

# How To Play
1. Go to [GamePlay](https://skylightanimation.github.io/kalimantan/playgame.html)
2. Set names of Player1 and Player2
3. Set Main Time and Turn Time
4. Start Game
